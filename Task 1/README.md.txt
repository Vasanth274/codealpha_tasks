 🎮 Hangman GAME (Streamlit Game)

A fun and interactive **Hangman game** built using **Python** and **Streamlit**, featuring a modern UI, animations, and a smooth user experience.



 ✨ Features

* 🎯 Interactive alphabet buttons
* ❤️ Lives system (6 chances)
* 🧠 Random word selection each game
* 🏆 Win & 💀 lose detection
* 🎉 Animations (balloons & snow on win)
* 🔄 One-click restart option
* 🎨 Beautiful gradient UI with custom CSS styling
* 📱 Responsive layout using Streamlit columns


 🛠️ Tech Stack

* Python 🐍
* Streamlit ⚡
* Random module 🎲
* HTML + CSS (for custom styling)

 🚀 How to Run the Project

 1. Clone the repository

git clone https://github.com/your-username/hangman-master.git
cd hangman-master

 2. Install dependencies

pip install streamlit

 3. Run the app

streamlit run app.py

🎮 How to Play

1. A hidden word is selected randomly.
2. Click on alphabet buttons to guess letters.
3. Each wrong guess reduces your lives (❤️ 6 total).
4. Guess the full word before lives run out to win!
5. Click **PLAY AGAIN** to restart anytime.
